# Father's Day Tribute Website

A responsive Father's Day tribute website built using HTML5 and CSS3. This project was created as a personal gift and learning project to celebrate cherished family memories while practicing front-end web development concepts.

## Overview

This website is dedicated to Father's Day and serves as a digital tribute expressing gratitude, love, and appreciation for fathers. It features a personalized greeting card, memory gallery, informative content about Father's Day, and a special conclusion page with music and visuals.

## Live Website

🔗 Website: https://fathersday-card.netlify.app/

## Features

- Personalized Father's Day greeting card
- Responsive design for mobile and desktop devices
- Interactive navigation links
- Family photo memories gallery
- Dedicated Father's Day information page
- Audio integration for a personalized experience
- Hover effects and custom styling
- Multi-page website structure
- Modern card-based user interface
